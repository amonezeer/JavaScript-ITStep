JSON - JavaScript Object Notation

Основні правила

  • Примітиви: дані типу "число" ( як ціле, так і дробове ), true,false,null,string(завжди у подвійних лапках "")
  • Масиви: перелік, поєднаний у квадратних дужках[]
  • Об'єкт: набір даних поєднаний у фігурних дужках{} Задається форматом "ключ": значення , де ключ завджи береться у подвійні лапки "", значення - залежно від типу даних.

Приклади правильного JSON:
123
"Hello"
[1,2,"3"]
{ "x":10, "y":20 }
{ "x":10, "y":"string", "arr": [1,2,3], "obj": {"w":30} }

AJAX - Async JavaScript and XML - технологія виконання запитів на сторонні ресурси (сервер - бекенд) без оновлення сторінки ( асинхронно )

Promise - об'єкт JavaScript, що є аналогом Task ( у С#), який відповідає за виконання асинхронних задач.
