Вступ до JavaScript

У JavaScript ';' вживається як роздільник (на відміну від ряду мов, де це завершення команди). Відповідно, не є обов’язковою і може замінюватися розривом рядка. У той же час, можлива неочікувана поведінка, коли одна команда, розділена на декілька рядків, перетворюється на різні. Для запобігання цього вважається традиційним: а) брати такі інструкції у дужки, б) не розривати рядки так, щоб команда була завершеною.

Неправильно: return 3
+ 2;

Неправильно: return
3+ 2;

Правильно: return (3
+ 2);

Правильно: return 3+
2;
